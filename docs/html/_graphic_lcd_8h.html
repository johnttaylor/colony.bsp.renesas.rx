GraphicLcd.h File Reference

Detailed Description

This file provides a basic interface for OKAYA 96x64 monochromo Graphic LCD. The RX62N CPU interfaces with display's STD7579 display controller via a a SPI bus (SPI0).

NOTE: This graphic driver is NOT THREAD SAFE. It it left to application to manage/handle/implement thread safety.

DO NOT include this file directly! Instead include the generic BSP interface - src/Bsp/Api.h.

uint8_t Bsp_GLcd_intialize ( void  )

This method initializes the graphics driver. This method must be called BEFOE any other methods in this interface. The initial font is determined by the USE_BSP_DEFAULT_FONT compiler switch (the default is the 5x7 font). The method returns 1 if successful; else 0 is returned.

uint8_t Bsp_GLcd_setFont ( BspGLcd_FontSelection_T  newfont)

This method sets the current font. All 'draw/write' operation after this call will use the new font. Previously drawn text is not effected. If the selected is not available then 0 is returned; else 1. When the method returns 0 - the current font is left unchanged.
