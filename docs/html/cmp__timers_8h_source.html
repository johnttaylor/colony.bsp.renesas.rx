40 /** This funciton initializes a 16bit timer channel for a Compare-Match
41  interrupt. In addition to calling this function, the application is
42  responsible for populating the corresponding IRQ/Vector for the CMT
43  interrupt. The CMT vectors are stored in the Relocatable vector table.
44  <pre>
45 
46  Interupt/Vector mappings:
47  ------------------------
48  CMT0 := CMTU0_CMI0, Irq#27 (0x70h)
49  CMT1 := CMTU0_CMI1, Irq#28 (0x74h)
50  CMT2 := CMTU1_CMI2, Irq#29 (0x78h)
51  CMT3 := CMTU1_CMI3, Irq#30 (0x7Ch)
52 
53  </pre>
54  This function can/should ONLY be called when the desired timer channel is
55  in the 'stopped' state. All timer channels after reset are considered to
56  be in the 'stopped' state.
57 
58 
59  @param channelNum CMT channel number. 0=CMT0, 1=CMT1, 2=CMT2, 3=CMT3
60  @param irqRateInHZ Desired frequency of the CMT interrupt.
61  @param pclkDivider PCLK divider/prescaler. 0= PCLK/8, 1=PCLK/32, 2=PCLK/128, 3=PCLK/512
62  @param irqPriority Interrupt Priority. 1=lowest, ... 15=highest
63 
64  @returns 1 if successful; else 0 is returned. Note: The error
65  checking is VERY LIMITED, i.e. it only checks validates the
66  range of arguments (except for irqRateInHZ which is not checked).
67  There is NO checking if the actual PCLK / divider / irqRate is
68  a 'valid' or accurate (i.e. a integer value).
69 
70  Example:
71  <pre>
72  // Sets CMT0 to generate an IRQ (CMI0) at every millisecond (1KHz) with
73  // an interrupt priority of 14. A PCLK divider of 8 is used.
74  Bsp_initCompareMatchTimer( 0, 1000, 0, 14 );
75 
76  </pre>
77  */
