88 /** This funciton initializes a 16bit timer channel for a Compare-Match
89  interrupt. In addition to calling this function, the application is
90  responsible for populating the corresponding IRQ/Vector for the CMT
91  interrupt. The CMT vectors are stored in the Relocatable vector table.
92 
93  Interupt/Vector mappings:
94  ------------------------
95  CMT0 --> CMTU0_CMI0, Irq#27 (0x70h)
96  CMT1 --> CMTU0_CMI1, Irq#28 (0x74h)
97  CMT2 --> CMTU1_CMI2, Irq#29 (0x78h)
98  CMT3 --> CMTU1_CMI3, Irq#30 (0x7Ch)
99 
100 
101  This function can/should ONLY be called when the desired timer channel is
102  in the 'stopped' state. All timer channels after reset are considered to
103  be in the 'stopped' state.
104 
105 
106  @param channelNum CMT channel number. 0=CMT0, 1=CMT1, 2=CMT2, 3=CMT3
107  @param irqRateInHZ Desired frequency of the CMT interrupt.
108  @param pclkDivider PCLK divider/prescaler. 0= PCLK/8, 1=PCLK/32, 2=PCLK/128, 3=PCLK/512
109  @param irqPriority Interrupt Priority. 1=lowest, ... 15=highest
110 
111  @returns true if successful; else false is returned. Note: The error
112  checking is VERY LIMITED, i.e. it only checks validates the
113  range of arguments (except for irqRateInHZ which is not checked).
114  There is NO checking if the actual PCLK / divider / irqRate is
115  a 'valid' or accurate (i.e. a integer value).
116 
117  Example:
118  // Sets CMT0 to generate an IRQ (CMI0) at every millisecond (1KHz) with
119  // an interrupt priority of 14. A PCLK divider of 8 is used.
120  Bsp_initCompareMatchTimer( 0, 1000, 0, 14 );
121  */
122 bool Bsp_initCompareMatchTimer( uint8_t channelNum, uint32_t irqRateInHZ, uint8_t pclkDivider, uint8_t irqPriority );
